Building a Modern React 19 Application with ArcGIS Maps

Introduction

Welcome to this comprehensive tutorial on building a modern web application using React 19 and ArcGIS maps! In this guide, we'll walk through the entire process of creating a web application that includes an interactive map component using Esri's ArcGIS JavaScript SDK and Calcite Design System.

What We'll Build

We'll create a simple web application with two main screens:

  1. A login screen (for demonstration purposes)
  2. A map screen that displays an interactive ArcGIS map with basic functionality

Who This Tutorial Is For

This tutorial is designed for beginner programmers who have:

  • Basic understanding of HTML, CSS, and JavaScript
  • Little to no experience with React or mapping technologies
  • Interest in building interactive web applications with maps

Technologies We'll Use

  • React 19: The latest version of the popular JavaScript library for building user interfaces
  • Vite: A modern, fast build tool that significantly improves the development experience
  • TypeScript: A superset of JavaScript that adds static typing
  • ArcGIS JavaScript SDK: Esri's powerful mapping platform
  • Calcite Design System: Esri's design system that provides consistent, accessible UI components

What You'll Learn

By the end of this tutorial, you'll understand:

  • How to set up a modern React application using Vite
  • How to integrate ArcGIS maps into a React application
  • How to implement basic user authentication
  • How to structure a React application following best practices
  • How to use Esri's Calcite components for a cohesive user interface
  • How to implement responsive design with a mobile-first approach
  • How to apply web accessibility best practices

Prerequisites

Before we begin, you'll need to have the following installed on your computer:

  • Node.js (version 18 or higher)
  • npm (usually comes with Node.js)
  • A code editor (we recommend Visual Studio Code)
  • A modern web browser (Chrome, Firefox, Edge, or Safari)

How This Tutorial Is Structured

We'll break down this project into manageable sections:

  1. Setting Up the Development Environment: Installing necessary tools and creating the initial project
  2. Understanding the Project Structure: Exploring the files and folders in our project
  3. Building the Login Page: Creating a simple authentication screen
  4. Integrating ArcGIS Maps: Adding and configuring the map component
  5. Styling with Calcite: Using Esri's design system for a professional look
  6. Making It Responsive: Ensuring our app works great on all devices
  7. Testing and Deploying: Preparing our application for production

A Note on Learning

Don't worry if some concepts seem challenging at first. We'll explain everything step by step, and by building this application, you'll gain practical experience with these technologies. Feel free to revisit sections as needed, and remember that learning to code is a journey!
